# 简历深度解析面试官 Agent

这是一个本地可运行的简历解析与模拟面试工具。它会根据简历文本、目标岗位和候选人级别，输出：

- 简历结构化解析
- 大厂进面概率判断
- 项目深挖和技术追问
- 不够资格进面时的改进建议
- 可复制到大模型中的 Agent 提示词

## 使用方法

直接用浏览器打开 `index.html`，粘贴简历文本后点击“开始解析并模拟面试”。

如果需要导入 PDF 或 Word 简历，请启动本地解析服务：

```powershell
python server.py --host 0.0.0.0 --port 8767
```

然后访问 `http://你的本机IP:8767/index.html`。

支持导入 `.pdf`、`.docx`、`.txt` 和 `.md` 文件。旧版 `.doc` 格式请先另存为 `.docx` 或 PDF。

## 文件说明

- `index.html`：应用入口
- `styles.css`：界面样式
- `app.js`：简历解析、评分、问题生成和建议逻辑
- `server.py`：PDF/DOCX/TXT 文件提取服务
- `agent-prompt.md`：可复用的 Agent 角色提示词

## 评分逻辑

当前版本使用规则引擎进行初筛，重点检查：

- 目标岗位关键技术栈
- 可深挖项目数量
- 量化指标数量
- 个人贡献表达
- 可验证证据
- 弱表达风险

如果需要接入真实大模型，可把 `agent-prompt.md` 作为系统提示词，把页面生成的“Agent提示词”和简历文本一起发给模型。
